## 收藏系统

**目标：** 将当前仅存路径字符串的 `favorites: string[]` 升级为结构化收藏系统。

**数据模型：**

```typescript
interface FavoriteItem {
  id: string;       // 唯一标识
  path: string;     // 文件/文件夹路径
  label: string;    // 显示名称
  type: 'file' | 'folder';
  color: string;    // 关联颜色标签
  addedAt: number;  // 添加时间戳
}
```

**存储：** localStorage + Tauri Store 双写，key 为 `favorites`。

**UI：**
- 侧边栏"我的收藏"动态列出所有收藏项，显示颜色圆点 + 名称
- 文件视图顶栏 ⭐ 按钮，点击添加/取消收藏，自动分配颜色
- 支持右键收藏项 → 取消收藏、在 Finder 中显示

**关联：** 收藏项颜色与路径颜色标签共用同一色值。

## 路径颜色标签

**目标：** 为每个路径绑定一个颜色，标签页通过 `currentPath` 获取颜色，实现视觉区分。

**数据模型：**

```typescript
type PathColorMap = Record<string, string>;
// key: 路径, value: 十六进制颜色值
```

**存储：** localStorage + Tauri Store，key 为 `path-colors`。

**颜色分配：**
- 预定义 10 种颜色（红橙黄绿青蓝靛紫粉灰）
- 基于路径 hash 取模自动分配，同一路径永远同色
- 右键标签页可手动更改颜色

**UI：**
- 标签页左侧显示颜色圆点
- 收藏列表项显示颜色标签
- 颜色跟随路径，拖拽标签页到其他窗口颜色不变
