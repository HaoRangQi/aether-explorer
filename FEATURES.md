# Aether Explorer — 功能规格明细

> 所有功能按 Tier 递进排列。每项含编号、功能名、详细说明、当前状态。
>
> 状态说明：`✅` = 已有可验收实现；`✅ 骨架` = UI / 入口已存在但能力未闭环；`⏳` = 待完成；`❌` = 明确不做或本期跳过。

---

## Tier 0 — 项目脚手架（5 项）

| 编号 | 功能 | 说明 | 状态 |
|------|------|------|------|
| 0.1 | Tauri 项目初始化 | 将现有 React+Vite 原型项目迁移到 Tauri v2，安装 CLI 和 API 依赖 | ✅ |
| 0.2 | Rust 后端骨架 | main.rs + lib.rs + Tauri 命令注册，集成 store/dialog/shell 插件 | ✅ |
| 0.3 | 构建配置 | macOS .app bundle、Info.plist、macOSPrivateApi、未签名本地构建配置（Developer ID 签名暂缓） | ✅ |
| 0.4 | 应用图标与元数据 | icns 图标、应用名称 Aether Explorer、bundle identifier | ✅ |
| 0.5 | 开发环境配置 | HMR 热更新、tauri dev、调试工具 | ✅ |

## Tier 1 — 窗口与外壳（8 项）

| 编号 | 功能 | 说明 | 状态 |
|------|------|------|------|
| 1.1 | 窗口管理 | 无边框窗口（decorations: false）、顶部拖拽区域、最小/最大/关闭按钮（侧边栏红黄绿三色） | ✅ |
| 1.2 | 窗口尺寸与位置 | 启动 1200×800、最小 900×600、记忆上次位置和尺寸到 Tauri Store | ✅ |
| 1.3 | 毛玻璃/Mica 效果 | CSS backdrop-filter 模糊效果，跟随主题透明度设置，计划后续换原生 NSVisualEffectView | ✅ |
| 1.4 | 浅色/深色/自动模式 | CSS 变量切换、跟随系统 Appearance、监听 prefers-color-scheme 变化 | ✅ |
| 1.5 | 强调色系统 | 6 种预设色 + 自定义 HEX 色值 + 8 套精品调色盘（Aurora/Sunset/Lavender/Forest/Ocean/Midnight/Ruby） | ✅ |
| 1.6 | 壁纸背景 | URL 导入自定义壁纸、Tauri 原生文件对话框上传本地图片（png/jpg/jpeg/gif/webp/bmp）、壁纸模糊度滑块 0-100px | ✅ |
| 1.7 | 字体选择 | 系统字体查询（queryLocalFonts API）、下拉菜单切换、实时预览 | ✅ |
| 1.8 | 中英文切换 | i18next 中/英双语，支持第三方语言包注册接口 registerLanguagePack，默认中文 | ✅ |

## Tier 2 — 真实文件系统接入（6 项）

| 编号 | 功能 | 说明 | 状态 |
|------|------|------|------|
| 2.1 | 目录读取 | Rust `std::fs::read_dir` 递归读取真实文件系统，替换 MOCK_FILES 假数据 | ✅ |
| 2.2 | 文件元数据 | 读取真实文件的大小（字节转可读格式）、修改时间、类型、权限位 | ✅ |
| 2.3 | MIME 类型检测 | 根据文件扩展名自动分类：图片（png/jpg/svg/webp）、视频（mp4/mov/mkv）、PDF、归档（zip/tar/gz）、代码、文本、文件夹等 | ✅ |
| 2.4 | 文件夹导航 | 双击进入子目录、返回上级（← 按钮）、目录层级追踪、导航历史 | ✅ |
| 2.5 | 路径面包屑 | 显示当前路径的各层级，每层可点击跳转、可点击编辑路径文本、可复制完整路径到剪贴板 | ✅ |
| 2.6 | 隐藏文件显示 | 过滤器开关：显示/隐藏以 `.` 开头的隐藏文件和文件夹（如 .gitignore、.DS_Store） | ✅ |

## Tier 3 — 三种视图模式（4 项）

| 编号 | 功能 | 说明 | 状态 |
|------|------|------|------|
| 3.1 | 列表视图 | 文件名/修改日期/大小/类型四列表头，点击表头排序（升/降/取消），4 档密度（宽松/标准/紧凑/极致），每行悬停显示更多操作按钮 | ✅ |
| 3.2 | 网格视图 | 自适应列数网格布局，文件夹/文件卡片展示，图片类型自动放大为 2×2 尺寸并显示缩略图预览，悬停上浮动画 | ✅ |
| 3.3 | 分栏视图 | Miller Columns 风格：多级目录横向展开，每列显示当前层级文件列表，点击文件夹在右侧新增列 | ✅ |
| 3.4 | 布局参数可调 | 网格项目宽度（100-400px）、高度（100-400px）、间距（4-64px）；分栏列宽度（200-600px）、项目高度（40-120px）；列表密度 4 档 | ✅ |
